Section 1: Simple Slope Calculation from Two Points

Definition: Slope m = (y₂ - y₁) / (x₂ - x₁). This measures the line's steepness.

Simple Example: Points (0, 0) and (2, 4). m = (4 - 0) / (2 - 0) = 4/2 = 2.

Another Example: Points (1, 3) and (5, 11). m = (11 - 3) / (5 - 1) = 8/4 = 2.

Section 2: Determine Slope from a Graph

Method: Pick two points on the line, then use the slope formula.

Example: Line passing through (0,1) and (3,7). Slope = (7-1)/(3-0) = 6/3 = 2.

Tip: For graphs, count the rise and run visually.

Section 3: Determine Graph from One Point and Slope

Method: Start at the point, use slope to find another point (rise/run), then draw the line.

Example: Point (1,2), slope 3. From (1,2), right 1 up 3 to (2,5). Draw line.

Another: Point (0,4), slope -1/2. Right 2 down 1 to (2,3).

Method: Plot the two points and connect them with a straight line.

Another: Points (-2,3) and (4,-3). Plot and connect.

Section 5: Slope-Intercept Form from Two Points

Method: Find slope m, then use one point to find b in y = mx + b.

Example: Points (2,3), (4,7). m=(7-3)/(4-2)=2. Using (2,3): 3=2*2 + b => b=-1. y=2x-1.

Another: (0,5), (3,2). m=(2-5)/(3-0)=-1. b=5. y=-x+5.

Section 6: Slope-Intercept from One Point and Slope

Method: Plug into y = mx + b to solve for b.

Example: Point (3,4), m=2. 4=2*3 + b => b=-2. y=2x-2.

Another: Point (1,-1), m=-3. -1=-3*1 + b => b=2. y=-3x+2.

Section 7: Standard Form of a Line

Definition: Ax + By = C, where A, B, C are integers, A ≥ 0.

Example: From y=2x+3: -2x + y = 3 or 2x - y = -3 (standardize to 2x - y = -3).

Another: From points (1,2), (3,6): m=2, y=2x. 2x - y = 0.

Section 8: Equation of Line Parallel to Another

Method: Use same slope m, then find b using a point.

Example: Parallel to y=3x+1, through (2,4). m=3. 4=3*2 + b => b=-2. y=3x-2.

Another: Parallel to y=-x+5, through (0,0). y=-x.

Interactive: Find Parallel Equation

Section 9: Equation of Line Perpendicular to Another

Method: Use negative reciprocal slope (-1/m), then find b using a point.

Example: Perp to y=2x-1, through (3,5). m=-1/2. 5=(-1/2)*3 + b => b=6.5. y=-0.5x+6.5.

Another: Perp to y=4x+2, through (1,0). m=-1/4. 0=-1/4*1 + b => b=0.25. y=-0.25x+0.25.

Section 10: Vertical and Horizontal Lines

Definition: Horizontal: y = k (slope 0). Vertical: x = h (undefined slope).

Example Horizontal: y = 4.

Section 11: Point-Slope Form

Definition: y - y₁ = m(x - x₁).

Example: Point (4,6), m=1. y-6=1(x-4).

From 2 Points: (1,2), (5,10). m=2. y-2=2(x-1).
