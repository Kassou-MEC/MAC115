Graphing Linear Equations

Point-Plotting, Quadrants, and Intercepts

Introduction to Graphing Linear Equations

A linear equation is an equation that forms a straight line when graphed on a coordinate plane. The standard form of a linear equation is:

Ax + By = C

where A, B, and C are constants, and x and y are variables.

Why Learn Graphing?

Graphing linear equations helps us:

  • Visualize relationships between variables
  • Solve systems of equations
  • Understand rates of change and slopes
  • Make predictions based on data trends
  • Apply mathematical concepts to real-world problems

Example: Real-World Application

The equation y = 5x + 10 could represent:

A phone plan that charges $10 monthly fee plus $5 per GB of data used.

Graphing this equation would show the total cost (y) for different amounts of data used (x).

The Coordinate Plane

The coordinate plane is a two-dimensional surface formed by two perpendicular number lines:

  • The x-axis (horizontal number line)
  • The y-axis (vertical number line)

The point where they intersect is called the origin, with coordinates (0, 0).

Quadrants

Quadrant I
(+, +)
Quadrant II
(-, +)
Quadrant III
(-, -)
Quadrant IV
(+, -)

The coordinate plane is divided into four quadrants by the axes.

Coordinate Plane

Points are located using ordered pairs (x, y).

Key Concepts

  • The x-coordinate indicates horizontal position (left/right)
  • The y-coordinate indicates vertical position (up/down)
  • Points on the x-axis have coordinates (x, 0)
  • Points on the y-axis have coordinates (0, y)
  • The origin is at (0, 0)

Plotting Points

Plotting points is the process of locating points on the coordinate plane using their coordinates (x, y).

How to Plot a Point

  1. Start at the origin (0, 0)
  2. Move horizontally to the x-coordinate (right for positive, left for negative)
  3. From there, move vertically to the y-coordinate (up for positive, down for negative)
  4. Mark the point at this location

Example: Plotting (3, 2)

Start at the origin (0, 0)
Move 3 units to the right (positive x-direction)
Move 2 units up (positive y-direction)
Mark the point at this location

Example: Plotting (-2, 4)

Start at the origin (0, 0)
Move 2 units to the left (negative x-direction)
Move 4 units up (positive y-direction)
Mark the point at this location

Finding Intercepts

Intercepts are points where a graph crosses the axes:

  • x-intercept: Where the graph crosses the x-axis (y = 0)
  • y-intercept: Where the graph crosses the y-axis (x = 0)

How to Find Intercepts

Finding x-intercept

To find the x-intercept, set y = 0 and solve for x.

Example: For 2x + 3y = 6
Set y = 0: 2x + 3(0) = 6
2x = 6
x = 3
x-intercept is (3, 0)

Finding y-intercept

To find the y-intercept, set x = 0 and solve for y.

Example: For 2x + 3y = 6
Set x = 0: 2(0) + 3y = 6
3y = 6
y = 2
y-intercept is (0, 2)

Special Cases

  • Horizontal lines (y = constant) have no x-intercept (unless y = 0)
  • Vertical lines (x = constant) have no y-intercept (unless x = 0)
  • Lines passing through the origin have both intercepts at (0, 0)

Practice Problem

Find the x-intercept and y-intercept of the equation 4x - 2y = 8.

x-intercept: Set y = 0

4x - 2(0) = 8
4x = 8
x = 2 → (2, 0)

y-intercept: Set x = 0

4(0) - 2y = 8
-2y = 8
y = -4 → (0, -4)

Graphing Linear Equations

To graph a linear equation, we can use two main methods:

  1. Point-plotting: Find several points that satisfy the equation and connect them
  2. Intercept method: Find the x and y intercepts and connect them

Point-Plotting Method

Example: Graph y = 2x + 1

Intercept Method

Find y-intercept: Set x = 0 → 2y = 6 → y = 3 → (0, 3)
