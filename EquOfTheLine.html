Understanding Slope, Graphing Lines, and Line Equations

Introduction to Slope and Linear Equations

Slope is a measure of the steepness of a line. It describes both the direction and the steepness of the line.

m = (y₂ - y₁) / (x₂ - x₁) = rise / run

where (x₁, y₁) and (x₂, y₂) are any two distinct points on the line.

Understanding Slope

Slope (m) is calculated as the ratio of the vertical change (rise) to the horizontal change (run) between two points on a line.

m = rise / run = (y₂ - y₁) / (x₂ - x₁)

Positive Slope

Line rises from left to right

Negative Slope

Line falls from left to right

Example: Calculating Slope

Find the slope between points (2, 3) and (5, 11)

Identify points: (x₁, y₁) = (2, 3), (x₂, y₂) = (5, 11)
Apply slope formula: m = (11 - 3) / (5 - 2)
Calculate: m = 8 / 3
The slope is 8/3, which means for every 3 units horizontally, the line rises 8 units vertically.

Special Cases

  • Horizontal lines have slope = 0 (no vertical change)
  • Vertical lines have undefined slope (no horizontal change)

Practice Problem

Find the slope between points (-1, 4) and (3, -2)

Identify points: (x₁, y₁) = (-1, 4), (x₂, y₂) = (3, -2)
Apply slope formula: m = (-2 - 4) / (3 - (-1))
Calculate: m = (-6) / (4) = -3/2
The slope is -3/2, which means for every 2 units horizontally, the line falls 3 units vertically.

Graphing Lines Given Two Points

To graph a line given two points:

  1. Plot both points on the coordinate plane
  2. Draw a straight line through both points
  3. Extend the line in both directions

Case 2: Finding Equation from Two Points

Find the equation of the line that passes through points (1, 2) and (3, 8)

Find slope: m = (8 - 2) / (3 - 1) = 6 / 2 = 3
Use point-slope form with point (1, 2): y - 2 = 3(x - 1)
Simplify: y - 2 = 3x - 3
Final equation: y = 3x - 1

Case 3: Finding Equation from Point and Slope

Find the equation of the line that passes through point (2, -1) with slope 3

Use point-slope form: y - y₁ = m(x - x₁)
Substitute values: y - (-1) = 3(x - 2)
Simplify: y + 1 = 3x - 6
Final equation: y = 3x - 7

Finding Slope from a Graph

To find the slope from a graph:

  1. Identify two clear points on the line
  2. Determine the rise (vertical change) and run (horizontal change) between them
  3. Calculate slope = rise / run

Case 1: Positive Slope

Find the slope of this line:

Identify two points: (1, 1) and (4, 3)
Rise = 3 - 1 = 2
Run = 4 - 1 = 3
Slope = rise/run = 2/3

Case 2: Negative Slope

Find the slope of this line:

Identify two points: (-2, 3) and (2, 1)
Rise = 1 - 3 = -2
Run = 2 - (-2) = 4
Slope = rise/run = -2/4 = -1/2

Case 3: Zero Slope

Find the slope of this line:

This is a horizontal line
Rise = 0 (no vertical change)
Slope = 0

Case 4: Undefined Slope

Find the slope of this line:

This is a vertical line
Run = 0 (no horizontal change)
Slope is undefined (division by zero)
